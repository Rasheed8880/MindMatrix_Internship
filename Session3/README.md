# Session 3 – Dynamic User Profile Console App

## Objective
Create a dynamic console application that displays formatted user profiles
with optional avatar boxes using Kotlin.

## Concepts Used
- Data classes
- Loops
- Functions
- String templates
- Conditional rendering

## How to Run

Compile:
kotlinc user_profile.kt -include-runtime -d user_profile.jar

Run:
java -jar user_profile.jar
